# Отчеты о тестировании

Эта папка содержит результаты запуска тестов.

## Файлы отчетов

### all-report.html
**Полный HTML отчет всех тестов**

Содержит результаты выполнения всех типов тестов:
- API тесты (30 тестов)
- Интеграционные тесты (18 тестов)
- Контрактные тесты (9 тестов)

**Итого: 57 тестов**

Отчет включает:
- Статус выполнения каждого теста
- Время выполнения
- Детальную информацию об ошибках (если есть)
- Сводную статистику

### locust-report.html
**Отчет о нагрузочном тестировании**

Содержит результаты нагрузочных тестов, выполненных с помощью Locust.

Включает:
- Статистику по типам запросов
- Время ответа (среднее, медиана, процентили)
- Количество запросов в секунду
- Процент успешных/неуспешных запросов
- Детализацию по эндпоинтам

**Параметры нагрузочных тестов:**
- Пользователей: 5
- Spawn rate: 1
- Время выполнения: 30 секунд
- Типы нагрузок:
  - Обычная нагрузка (SpringBootUser)
  - Негативные сценарии (SpringBootNegativeUser)
  - Скачки нагрузки (SpringBootSpikeUser)
  - Стресс-тестирование (SpringBootStressUser)

## Просмотр отчетов

Откройте файлы в любом веб-браузере для просмотра детальных результатов тестирования.
